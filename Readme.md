# 数当てゲーム

## ゲームの概要

特定の範囲の値を当てる。    
※特定の範囲の値とは1以上100以下の正数とする。

## 処理の流れ

1. 起動
2. 特定の範囲の値(乱数)uvを決める
3. 値ivの入力を促す
   * `q`を入力したらゲームを終了する
4. ivがuvと等しいか判定する
5. iv == uv ゲーム終了または2へ
6. iv > uv 「ivはuvより大きいです」とメッセージを表示後、3へ
7. iv < uv 「ivはuvより小さいです」とメッセージを表示後、3へ

## 略称について

* uv = unknown value
* iv = input value
