# 数当てゲーム

## ゲームの概要

特定の範囲の値を当てる。    
※特定の範囲の値とは1以上100以下の正数とする。

## 処理の流れ

1. 起動
2. 特定の範囲の値(乱数)unを決める
3. 値inの入力を促す
   * `q`を入力したらゲームを終了する
4. inがunと等しいか判定する
5. in == un ゲーム終了または2へ
6. in > un 「inはunより大きいです」とメッセージを表示後、3へ
7. in < un 「inはunより小さいです」とメッセージを表示後、3へ

## 略称について

* un = unknown number
* in = input number
